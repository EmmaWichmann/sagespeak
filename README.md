# Sage & Speak 🌸

A speech affirmation iOS app built for people who stutter or struggle with speech anxiety. Designed to be a calming, beautiful daily practice tool.

---

## What It Does

- **🌸 40 Affirmations** across 5 categories — Confidence, Fluency, Courage, Self-Worth, and Calm
- **💗 Favorites** — heart any affirmation to save it to your personal collection
- **🔥 Streak Tracking** — tracks your daily practice streak to build consistency
- **📅 Week Tracker** — visual week view showing which days you practiced
- **🍃 Therapy Tab** — a dedicated space to track therapy sessions including:
  - Weekly focus/goals text box
  - Mood logging (Great → Really Hard)
  - Session notes with free-form journaling
  - Past sessions log
- **📊 Progress Tab** — all-time stats, favorites count, sessions logged

---

## Tech Stack

- Swift
- SwiftUI
- UserDefaults for persistent storage
- No external dependencies
