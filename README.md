# Sage & Speak

A gentle affirmation and reflection app built for people working through a stutter or speech impediment, made from my own experience with one.

## Why this is a web app and not a Swift app

Sage & Speak actually started as a native iOS app, built in SwiftUI. That version worked well for months. Recently, building it in Xcode started crashing my computer, and even creating a brand new test project hit the same issue. Rather than wait on fixing that, I rebuilt the app as a web version instead, using plain HTML, CSS, and JavaScript.

Being honest, this switch was driven by that limitation, not by the web version being strictly better. I still want the native app to exist, and I plan to go back to it once the Xcode issue is resolved. But it also turned into a good opportunity.

**Accessibility.** A web app runs in any browser, on any device, no App Store download required. Someone can open a link and start using it in seconds, on their phone, their laptop, whatever they have in front of them. For a tool meant to support someone in a hard moment, that low barrier matters.

**It fits how I think.** I have a background in the life sciences, and I am building toward work at the intersection of medicine and engineering. Medicine is about understanding a person's needs closely and designing around them with care. Engineering is about building the actual system that meets those needs, reliably, simply, and in a way that can be improved over time. This project lets me practice both sides of that at once, the empathy driving the feature, and the code making it real.

**Iteration speed.** Web technology let me keep building without needing Xcode to cooperate, so the project could keep moving instead of stalling out.

## What it does

- Daily affirmations across five categories, Confidence, Fluency, Courage, Self-Worth, and Calm
- A streak and progress tracker to build a consistent practice
- A Favorites tab to save the affirmations that matter most
- A Therapy Space tab for weekly focus setting and session journaling
- All data saves locally in the browser, so your progress is still there next time

## Built with

HTML, CSS, and JavaScript. No frameworks, no build tools, just the fundamentals, on purpose. Part of what I am building right now is real technical fluency underneath any AI-assisted workflow, and this project is one place I am practicing that.
